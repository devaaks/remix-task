## Getting Started

To get started with this project, follow the steps below:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/devaaks/remix-task
   cd remix-assignment
   ```

2. **Running the Project**:

    #### Install dependencies
    ```bash
    npm install
    ```

    #### Run the dev server:

    ```shellscript
    npm run dev
    ```
